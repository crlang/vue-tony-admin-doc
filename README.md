## 开始编辑

```bash
# 克隆本仓库
$ git clone https://github.com/crlang/vue-tony-admin-doc.git

# 安装依赖
$ npm install
# 或者使用 pnpm
$ pnpm i

# 启动开发服务器
$ pnpm dev
```
